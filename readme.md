Chat'ouille - Version 2

- Gestion de plusieurs Chatroom
- Affichage des flux Twitter en rapport avec le nom du salon
- Gestion des pseudos (pseudo unique, ...)
- Génération de pages statics reprennant les conversations
- Affichage sur la page d'accueil => optimisation SEO


Technologies

- NODE.JS
- Framework Flatiron
- Socket.IO
- Plates
- Plug-in "static" pour Flatiron
- HTML / CSS / Javascript (JQuery, JQuery UI)
